# Paper Analyzer

## Description

This project is a multi-agent system for analyzing academic papers.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/LorenzoRidi/PaperAnalyzer.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd PaperAnalyzer
    ```
3.  Install the dependencies using Poetry:
    ```bash
    poetry install
    ```

## Usage

To run the agent, execute the following command:

```bash
poetry run python PaperAnalyzerAgent/agent.py
```
